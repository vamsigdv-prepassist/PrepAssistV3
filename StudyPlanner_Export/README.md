## How to Integrate
1. Drag and drop `StudyPlanner.tsx` into your `src/components/` directory (or directly into your `src/app/` router if it's a dedicated page).
2. If dropping it into an existing Next.js App Router layout, ensure it retains the `"use client";` directive at the top of the file since it handles intensive DOM events and local states.
3. **Importing:**
```tsx
import StudyPlanner from '@/components/StudyPlanner';

export default function MyDashboardRoute() {
```

- **Multi-Day Calculations:** Visual spans are handled dynamically in the `renderTaskBars()` method, which calculates grid placement using pure Javascript `Date` mathematics overlaid on CSS negative margins (`-mx-3`).
